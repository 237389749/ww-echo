# ww-echo

一个基于图像识别的鸣潮声骸强化自动化工具，从 [ok-ww](https://github.com/ok-oldking/ok-wuthering-waves) 剥离，仅保留声骸相关功能，基于 [ok-script](https://github.com/ok-oldking/ok-script) 开发。

*通过 Windows 接口模拟用户进行操作，无内存读取、无文件修改*

## 功能

### 批量强化声骸 (EnhanceEchoTask)

自动将背包中的 0 级声骸强化至满级并调谐，根据用户设定的规则判断保留或丢弃。

**使用步骤：**
1. 游戏内按 `B` 打开背包，切换到声骸标签页
2. 使用过滤器筛选需要强化的声骸（如：按等级从低到高排序）
3. 启动任务

**强化规则（均可配置）：**

| 规则 | 默认值 | 说明 |
|------|--------|------|
| 必须有双爆 | 开启 | 声骸最终必须同时拥有暴击和暴击伤害，剩余孔位不足以凑齐则丢弃 |
| 双爆前全有效词条 | 开启 | 第一个双爆词条出现之前，前面所有词条必须在有效列表中 |
| 首条双爆 ≥ | 6.9 | 第一条出现的暴击或暴击伤害需满足此数值（爆伤÷2计算） |
| 双爆总计 ≥ | 13.8 | 暴击 + (暴击伤害÷2) ≥ 此数值 |
| 有效词条 ≥ | 3 | 满级时需要的有效词条数量，剩余孔位无法凑齐则提前丢弃 |
| 第一条必为有效词条 | 开启 | 第一个副词条必须在有效词条列表中且符合数值要求 |
| 有效词条 | 暴击/爆伤/攻击% | 定义哪些属性算有效（可多选） |

**可选有效词条：** 暴击伤害、暴击、攻击百分比、生命百分比、防御百分比、攻击、生命、防御、共鸣效率、普攻伤害加成、重击伤害加成、共鸣解放伤害加成、共鸣技能伤害加成

**操作说明：**
- 符合条件 → 自动上锁（`C 键`）+ 截图保存至 `screenshots/success/`
- 不符合条件 → 自动弃置（`Z 键`）+ 截图保存至 `screenshots/failed/`
- 可配置成功后暂停任务（`Pause after Success`），用于手动复核

### 批量修改声骸主属性 (ChangeEchoTask)

自动将背包中声骸的主属性修改为目标属性（数据重构）。

**目标属性选项：** 攻击、暴击伤害、暴击、生命、防御、共鸣效率、冷凝/热熔/导电/气动/衍射/湮灭伤害加成

## 运行环境

- **操作系统：** Windows
- **Python：** 3.12
- **游戏分辨率：** 16:9（最低 1600x900）
- **游戏语言：** 简体中文 / 繁体中文
- **帧率：** 稳定 60 FPS

## 从源码运行

```bash
pip install -r requirements.txt --upgrade
python main.py        # Release 版本
python main_debug.py  # Debug 版本（显示识别框等调试信息）
```
